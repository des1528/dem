# Салон красоты — Сессия 1 (учебный проект для ДЭ-2026)

## Зависимости

```
pip install mysql-connector-python pillow pandas openpyxl tkcalendar
```

## Подготовка БД

1. В MySQL Workbench создайте БД `salon` и выполните `my.sql` из ресурсов.
2. Заполните справочник пола:
   ```sql
   INSERT INTO Gender VALUES ('М','Мужской'),('Ж','Женский');
   ```
3. Поправьте параметры подключения в [db.py](db.py) (host/user/password).
4. Импортируйте исходные данные: `python import_data.py`.

## Запуск

```
python main.py
```

## Режим администратора

Кнопка «Войти как админ» в правом верхнем углу. Код: **0000**.

В режиме админа доступны: добавление/редактирование/удаление услуг,
работа с фотографиями, запись клиента на услугу, окно «Ближайшие записи».

## Файлы

- [db.py](db.py) — обёртка над MySQL (`fetch`, `query`).
- [main.py](main.py) — всё приложение в одном файле.
- [import_data.py](import_data.py) — однократная заливка ресурсов в БД.
- [GUIDE.md](GUIDE.md) — подробное пошаговое объяснение для подготовки к экзамену.
