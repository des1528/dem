## Технические уроки и приёмы

### 3НФ в наших схемах

Все БД приведены к 3НФ. Главный приём — **выделение справочника** (`Role`, `Category`, `Manufacturer`, `Supplier`, `Gender`, `PickupPoint`): вместо хранения текстового значения и его описания в одной таблице, разносим в две и связываем через FK.

Связь many-to-many реализуется через **таблицу-связку** (`OrderItem`, `AttachedProduct`, `ServicePhoto`). Это устраняет нарушение 1НФ «список через запятую».

### Защита от GC у изображений

Самая частая ошибка с tkinter / CTk:

```python
# плохо — картинка пропадёт через секунду
ctk.CTkLabel(parent, image=ctk.CTkImage(...)).pack()

# хорошо — держим ссылку
photo = ctk.CTkImage(...)
lbl = ctk.CTkLabel(parent, image=photo)
lbl.image = photo  # ← вот эта строка
lbl.pack()
```

### Реактивный поиск/фильтр

```python
search_var = ctk.StringVar()
search_var.trace_add("write", lambda *_: reload())
```

`trace_add("write", ...)` — встроенный механизм tkinter, наследуется в CTk. Срабатывает на каждое нажатие клавиши.

### Запрет двух модалок

```python
edit_win = None  # глобально

def open_edit(...):
    global edit_win
    if edit_win and edit_win.winfo_exists():
        edit_win.lift(); return
    edit_win = ctk.CTkToplevel(...)
```

`winfo_exists()` работает и с уже уничтоженным окном — возвращает `0`.

### Сортировка вручную (для расширяемости)

```python
rows = sorted(rows, key=lambda r: float(r["Cost"]),
              reverse=sort_var.get().startswith("По убыванию"))
```

В ТЗ Салона прямо сказано: не использовать сортировку TreeView, потому что планируется расширение функционала. `sorted(..., key=)` — это легко расширяется добавлением `tuple` в ключ.

### CTk-шаблон модалки

```python
w = ctk.CTkToplevel(root); w.title("..."); w.geometry("...")
w.configure(fg_color=BG_MAIN); w.transient(root); w.lift(); w.focus()
hd = ctk.CTkFrame(w, fg_color=BG_MAIN, height=56, corner_radius=0); hd.pack(fill="x"); hd.pack_propagate(False)
ctk.CTkLabel(hd, text="...", font=F_HEAD, text_color=BG_BRAND).pack(side="left", padx=20)
ctk.CTkFrame(w, fg_color=BG_ACCENT, height=3, corner_radius=0).pack(fill="x")
# ... содержимое ...
ctk.CTkButton(w, text="Сохранить", fg_color=BG_ACTION, ...).pack(fill="x", padx=24, pady=14)
```

Это шаблон повторяется в трёх модалках salon_ctk — стоит его выучить.
