# Салон красоты — CustomTkinter edition

Та же функциональность, что в [../salon/](../salon/), но интерфейс на **CustomTkinter** с современным дизайном и закруглёнными виджетами. Соблюдены критерии стиля М2:

| Цвет / шрифт | Где применён |
|---|---|
| `#FFFFFF` основной фон | окно и список карточек |
| `#7FFF00` доп. фон | шапка, рамки полей ввода, иконка-плейсхолдер |
| `#00FA9A` целевые кнопки | «Войти как админ», «+ Услуга», «Сохранить», «Записать» |
| `#2E8B57` карточка скидки > 15% | фон карточки услуги |
| `#E64545` срочный таймер | «осталось < 1 часа» в «Ближайших записях» |
| Times New Roman | весь текст |

## Установка

```powershell
pip install customtkinter mysql-connector-python pillow tkcalendar
```

Базу данных можно взять из соседнего проекта (общая БД `salon` в MySQL) или развернуть локально:
```powershell
mysql -u root -proot < init_demo.sql
```

`db.py` и `init_demo.sql` — копии из [../salon/](../salon/), чтобы проект был самодостаточным.

## Запуск

```powershell
python main.py
```

Код режима администратора: `0000`.

## Чем отличается от оригинального `salon/`

- `CTk()` вместо `Tk()`, `CTkFrame/CTkLabel/CTkButton/CTkEntry/CTkOptionMenu` вместо tk-виджетов.
- `CTkScrollableFrame` вместо ручной обвязки `Canvas + Frame + Scrollbar`.
- `CTkImage` вместо `ImageTk.PhotoImage` (масштабируется на high-DPI без размытия).
- Закруглённые углы, hover-эффекты, тонкие рамки.
- Кнопки «срочно» и «> 15%» — цветными бейджами.
- Палитра и шрифт строго по М2.

Сам бизнес-функционал и SQL — идентичны оригиналу.
