# ДЭ-2026 — учебные проекты на Python

Подготовка к демонстрационному экзамену по специальности 09.02.07 «Информационные системы и программирование».

## Проекты

| Папка | Что это |
|---|---|
| [salon/](salon/) | Салон красоты (Сессия 1) — Tkinter + MySQL |
| [shoes/](shoes/) | Магазин обуви (Модуль 1) + стили М2 |
| [pets/](pets/) | Товары для животных — captcha, библиотека SF2022User01Lib, 10 unit-тестов |
| [variativ/](variativ/) | Руководство пользователя (Word, python-docx) |
| [Demo2026/](Demo2026/) | Эталонное WPF/C# приложение (магазин Гены Букина) |

Каждый проект содержит:
- `main.py` / `*.cs` — исходники
- `schema.sql` — схема БД (3НФ)
- `import_data.py` — однократная загрузка ресурсов
- `PLAN.md` — план реализации
- `GUIDE.md` — подробное пошаговое объяснение для экзамена
- `README.md` — быстрый старт

## Зависимости
```
pip install mysql-connector-python pillow openpyxl pandas python-docx tkcalendar
```

## Что не залито
- `*.mp4` — видеоуроки (превышают лимит GitHub в 100 МБ)
- `_extracted/` — распакованные .rar (есть в исходной папке `Подготовка к ДЭ-2026/`)
- `bin/`, `obj/`, `.vs/` — артефакты сборки WPF
- `__pycache__/`
