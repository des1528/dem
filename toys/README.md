# МирИгрушек — ДЭ-2026 Вариант №4

Магазин игрушек ООО «МирИгрушек». CustomTkinter + MySQL.

## Быстрый старт (Windows)

1. Создать виртуальное окружение и активировать:
   ```powershell
   python -m venv .venv
   .\setexecution.ps1
   .\.venv\Scripts\Activate.ps1
   pip install -r requirements.txt
   ```
2. Открыть MySQL Workbench, выполнить скрипт `init_demo.txt` (он создаёт БД `toysdb`, таблицы по 3НФ и заполняет тестовыми данными из xlsx).
3. Проверить параметры подключения в `db.py` (по умолчанию `root` / `root` на `localhost`).
4. Запустить:
   ```powershell
   python main.py
   ```

## Структура

```
toys/
├── main.py            — приложение (CustomTkinter, все окна)
├── db.py              — обёртка MySQL (conn / fetch / query)
├── init_demo.txt      — SQL: схема 3НФ + тестовые данные из xlsx
├── requirements.txt   — зависимости
├── setexecution.ps1   — обход политики PowerShell для активации venv
├── make_resources.py  — генератор placeholder-картинок
└── resources/         — фото 1.jpg–10.jpg, picture.png (заглушка), icon, logo
```

## Соответствие модулям

- **М1**: схема `init_demo.txt` — 11 таблиц в 3НФ с FK, тестовые данные строго из xlsx (10 товаров, 10 пользователей, 10 заказов, 36 пунктов выдачи).
- **М2**: палитра `#FFFFFF` / `#F5DEB3` / `#DEB887`, шрифт Arial, логотип на главной форме, иконка приложения, авторизация + гость, ФИО пользователя в правом верхнем углу, фон строки `#FFDEAD` при скидке > 17%, цена со скидкой — перечёркивание красным + новая цена чёрным, нет на складе — голубая строка.
- **М3**: последовательный UI с кнопкой «Назад», окна добавления/редактирования товара, реальный поиск + сортировка + фильтр (комбинируются), запрет двух окон редактирования, ID-артикул read-only при редактировании, картинка ресайзится до 300×200, проверка цены и количества на отрицательность, удаление невозможно если товар в заказе.
- **М4**: окно «Заказы» доступно менеджеру и администратору, CRUD заказов — только администратор, поля: артикул, статус (выпадающий), адрес пункта выдачи, дата заказа, дата выдачи.
